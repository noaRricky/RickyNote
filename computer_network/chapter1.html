# 第一章 计算机网络和因特网

# 什么是因特尔网?

  • 从具体构成描述: 由主机和端系统, 因特尔网运营商组成的
  • 从服务描述: 分布式应用程序

# 什么是协议?

  • 发送和接受报文时所采用的约定动作

# 网络边缘

端系统可以称作主机，(包括手机, PC, Linux盒), 可以分为Server和Client

# 接入网的几种环境

  • 家庭接入
  • 企业接入
  • 广域无线接入： 3G和LTE

# 物理媒体

  • 可以划分为: 导引型媒体和非导引型媒体

# 网络核心

  • 交换策略可分为: 分组交换和电路交换，分组交换比电路交换更有效
  • 网络构成，假象经过ISP 5次迭代更新

# 分组交换的问题

  • 时延
    • 处理时延
    • 排队时延
    • 传输时延
    • 传播时延
  • 丢包
  • 吞吐量
