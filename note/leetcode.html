# 862. Shortest Subarray with Sum at Least K

  • 使用方法: 滑动窗口, 单调队列, 前缀和
  • 如果只使用单调队列时候，会发现因为数组中存在负数，当增大left的数值即缩小窗口大小，窗口内的数值和不是单调递减的，因此考虑引入单调性
  • 当固定窗口右侧搜索左侧窗口位置时候，如果前缀和数组中值递减，那么可以忽略左侧较大的数值，因为在位置上右侧数值的位置值较大，数值较小，得到的窗口值较大，所以左侧值可以直接忽略，故维护一个单增的队列
  • 当固定窗口左侧，考虑右侧时候，如果前缀和数值递增(x1 < x2, P[x1] < P[x2])，那么我们一定选择前一个组合，不需要x跟窗口左边界组合，所以在计算窗口值时候，可以放心popleft左侧数值
  • 参考: https://leetcode-cn.com/problems/shortest-subarray-with-sum-at-least-k/solution/he-zhi-shao-wei-k-de-zui-duan-zi-shu-zu-by-leetcod/

# 二分算法需要注意的边界条件

def leftBinarySearch(arr: List[int], target: int) -> int:
    left = 0
    right = len(arr)

    while left < right:
        mid = (left + right) >> 1
        if arr[mid] < target:
            mid = left + 1
        else:
            mid = right

    return mid

def rightBinarySearch(arr: List[int], target: int) -> int:
    left = 0
    right = len(arr)

    while left < right:
        mid = (left + right) >> 1
        if arr[mid] <= target:
            mid = left + 1
        else:
            mid = right
  • 左边界二分搜索得到的结果位置为 target 对应位置或者比 target 大的第一个数据的位置
  • 注意: 左边界二分搜索的位置可能为数组长度
  • 右边界二分搜索得到的是 target 下一位或者比 target 大的第一个数据的位置
  • 注意: 右边界二分搜索得到0时候，表示不存在该数
